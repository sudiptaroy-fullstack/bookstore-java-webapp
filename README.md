# 📚 BookStore Web Application

An online bookstore web application developed using JSP, Servlets, JDBC, and MySQL featuring authentication, cart, wishlist, search, and dynamic stock management.

---

## 🚀 Features

- 🔐 User Authentication (Login / Signup)
- 📖 Browse books by categories
- 🔍 Search books
- 🛒 Add to Cart
- ❤️ Wishlist functionality
- 📦 Order placement
- 📊 Dynamic stock display

---

## 🛠 Tech Stack

- Java (Core Java)
- JSP & Servlets
- JDBC
- MySQL
- JSTL
- HTML, CSS, Bootstrap

---

## 📂 Project Structure

- `controller` → Servlets
- `dao` → Database logic
- `model` → Java classes
- `jsp` → Views
- `assets` → CSS, JS, Images

---

## ⚙️ How to Run

1. Import project into NetBeans
2. Configure MySQL database
3. Update DB credentials in `DBUtil.java`
4. Run using Apache Tomcat
